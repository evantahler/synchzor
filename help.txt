Synchzor Help:

Commands:
  list
  add
  remove
  synch
  sync
  all
  reset
  remote_clean
  remote_reset
  setup

list: list the folder monitored on this machine and their settings
    inputs: none

add: add a new local folder to synch
    inputs: (optional)local_folder=Path/To/Folder/ password=RemoteServerPassword host=yourhost.com (optional)remote_path=synchzor/your/remote/folder

remove: stop synching a local folder.  Content will be maintained locally and on the server.
    inputs: local_folder=Path/To/Folder/

synch: sync the local folder to the remote server
    inputs: (optional)local_folder=Path/To/Folder/

sync: alias for synch for you spelling purists
    inputs: (optional)local_folder=Path/To/Folder/

all: will synch local folders.  This might be good to run on a schedule
    inputs: none

reset: reset all local settings, forget all synched folders.  local and remote content will be maintained.
    inputs: none

remote_clean: remove all files and records from the server for this synched folder
    inputs: local_folder=Path/To/Folder/

remote_reset: removes all files from the remote server, and then pushes the local copy {remote_clean + synch}
    inputs: local_folder=Path/To/Folder/
